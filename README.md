# m328v6上位机APP  

这个是针对一乐论坛 M8V6数控电子负载 的 升级版 M328V6数控电子负载 的新版上位机。  
[M328V6数控电子负载](https://www.yleee.com.cn/thread-90734-1-1.html)  

## 特性：  
1. 支持Android, iOS, Windows, Mac OS, Linux   
2. 支持多语种   
3. 支持OTG有线连接USB串口和无线连接蓝牙串口   
4. 支持全功能的控制下位机   
5. 支持放电曲线显示   
6. 支持保持屏幕常亮
